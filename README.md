# 🎯 WorkPilot - Project Management App – Frontend

This is the **frontend** of a real-time project management app built with **React.js (Vite)**, **Tailwind CSS v3**, and **Socket.io**.  
The app offers a smooth, responsive UI for managing workshops, projects, and tasks with real-time collaboration.

## 🛠 Tech Stack

- ⚛️ React.js (Vite)
- 🎨 Tailwind CSS v3
- 💅 Shadcn/UI + Magic UI
- 🧠 Context API for state management
- 🔌 Socket.io (client)
- 🖼 Lucide Icons

---

## 🚀 Features

- 📦 **Authentication**
  - Login/Register via email or Google
  - JWT-based secure auth

- 🧑‍🤝‍🧑 **Workshop Management**
  - Create, update, delete, and join workshops
  - Invite members by code

- 📁 **Projects & Tasks**
  - Nested task management inside projects
  - Real-time updates via sockets

- 🧑‍💻 **Role-Based Access**
  - Dynamic UI updates based on user role (Owner, Admin, Member)

- 🌙 **Modern UI**
  - Responsive layout using Tailwind
  - Styled with Shadcn components & Magic UI animations

- 💥 **Error Handling**
  - Global error boundaries
  - Optimistic UI updates

## 📁 Folder Structure
``` bash
src/
├── assets/            # Static assets (images, svgs)
├── components/        # Reusable UI components
├── context/           # Context API (auth, socket, etc.)
├── hooks/             # Custom React hooks
├── lib/               # API utilities, socket client, etc.
├── pages/             # Page-level components (routes)
├── App.jsx            # Root component
├── ErrorBoundary.jsx  # Global error handler
└── main.jsx           # Entry point

```
